link para acessar a aplicação:

[Meu primeiro deploy](https://meu-primeiro-deploy-665.herokuapp.com/)

## 🏃‍♀️ Como rodar o projeto na sua maquina?

---
### 1. Faça um clone do projeto:
```
git clone https://github.com/madruga665/meu-primeiro-deploy-heroku.git
```

### 2. Instale as dependências:

```
npm install
```

### 3. Rode a aplicação utilizando o comando:

```
npm run dev
```
